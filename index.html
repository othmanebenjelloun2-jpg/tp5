📚 Rappel du dernier cours de Web

⏳ Chargement de MediaPipe…
💡
Boîte d'info : ouverte = main ouverte, cachée = poing fermé. C'est ce qu'on contrôle avec display: flex / display: none.

1. HTML — la structure

HTML (HyperText Markup Language) sert à structurer le contenu : titres, paragraphes, listes, images, liens.

<h1>Titre principal</h1>
<p>Un paragraphe avec un <a href="...">lien</a>.</p>

Chaque balise a un rôle sémantique : <header>, <nav>, <main>, <footer>…

2. CSS — le style

CSS (Cascading Style Sheets) habille la page : couleurs, polices, espacements, mises en page (Flexbox, Grid), animations.

h1 {
  color: #0066cc;
  font-size: 2rem;
  text-align: center;
}

La propriété display est centrale : block, inline, flex, grid, ou none pour cacher un élément — c'est exactement ce qu'on fait avec la main !

3. JavaScript — le comportement

JS rend la page interactive : on réagit aux clics, au scroll, aux touches du clavier, et on modifie le DOM en direct.

const btn = document.querySelector('button');
btn.addEventListener('click', () => {
  alert('Salut !');
});

4. Le DOM

Le DOM (Document Object Model) est l'arbre des éléments HTML que JS peut lire et modifier : document.getElementById(), element.style.display = 'none', etc.

5. Les événements & les bibliothèques externes

On peut charger une bibliothèque depuis un CDN avec une balise <script>. C'est ce qu'on fait pour MediaPipe, la bibliothèque de Google qui détecte les mains, le visage et la pose directement dans le navigateur.

Pas besoin d'installer Python ou de serveur : tout tourne en JS dans ton navigateur. 🎉

6. Récap MediaPipe

MediaPipe nous renvoie 21 points par main (les "landmarks"), avec leurs coordonnées x, y, z.

On compare la position des bouts de doigts à celle des articulations pour savoir si la main est ouverte ou fermée. Simple !
